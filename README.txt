** Interdroid Versioned Databases **
------------------------------------

This distribution contains version 0.1 of the User Interface
portions of the Interdroid Versioned Database system.


What Is interdroid-vdb-io?
----------------------------

This project aims to make it easy to control the underlying sharing features
of the interdroid versioned database project

Requirements:
-------------

You must have an SLF4J project in your workspace. We suggest using our
implemenation available here:

git@github.com:interdroid/slf4android.git

Additionally you must have an interdroid-vdb and interdroid-vdb-avro
project in your workspace.  This project makes use of external
software available in the interdroid-vdb project.  See the external
directory in that project for licenses for this software.
